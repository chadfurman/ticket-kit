# TD-0007 · Theme the board for light mode

## Why

The neon-on-black board is great at night; a light theme would help in daylight
and in screenshots for docs.

## What

- A light palette behind a `prefers-color-scheme` media query (or a toggle).

## Acceptance

- [ ] The board is readable in light mode.
- [ ] No new runtime dependencies.
